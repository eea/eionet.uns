README for the REPORTNET UNS

Requirements:
    - Sun's J2SE Development Kit 1.4.2      (download from http://java.sun.com/j2se/1.4.2)
    - Apache Tomcat 5.0.x                      (download from http://jakarta.apache.org/tomcat)
	- MySQL database server 4.1.x, x>08   (download from http://www.mysql.com)
	  	Note: WDB can not work with older MySQL versions than 4.1
	- Apache Maven 1.0 or 1.1                 (download from http://maven.apache.org)
		Note: Meven is used for the WDB automatic build & installation
